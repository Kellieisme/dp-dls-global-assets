# Security Policy

## Supported Versions

The following release versions are
currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security issue in this repository, please report it.

This will allow us to assess the risk, and make a fix available before we add a bug report to the repository.

Thanks for helping make this repository safe for everyone!
